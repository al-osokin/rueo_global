# rueo_global docs (from repo root)

Эта папка содержит копии/снимки markdown-документов, которые исторически лежали в корне репозитория `~/rueo_global`.

Цель: держать «память проекта» и рабочие планы в одном месте (`memory-bank/`), не ломая ничего в репозитории. Поэтому пока это **копии** (оригиналы в корне остаются).

## Где что лежит

- `root-md-snapshots/` — снимки файлов из корня репо (Agents/TASK/PLAN/…)

## Рекомендованная структура дальше

- `memory-bank/README.md` — оглавление (что читать новичку, что актуально сейчас)
- `memory-bank/docs/` — долговечные гайды и архитектурные заметки
- `memory-bank/tasks/` — задачи/планы (можно постепенно переносить из `root-md-snapshots/`)
- `memory-bank/archive/` — устаревшее/историческое (например, полный лог Agents)
