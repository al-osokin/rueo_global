## ADDED Requirements
### Requirement: Vite-based Quasar CLI
The frontend build system SHALL use the Quasar CLI with Vite for development and production builds.

#### Scenario: Build uses Vite CLI
- **WHEN** a developer runs `quasar dev` or `quasar build`
- **THEN** the Vite-based Quasar CLI is used to compile the frontend.
